Variables

Es el bloque más básico.

Primero declarar, y asignar valores más tarde, o modifcar valores existentes.

Nombres de las variables:

    let, const, var

    var es la manera tradicional de declarar variables (desde que se creó javascript en 1995). Pero en el 2015 salió ES6, y var fue reemplazado por let y const.

    const es una constante, no se puede reasignar. Y tampoco podemos declararla sin asignarle ningún valor.

    Una práctica común es asignar todas las variables como const, y luego si es necesario poder cambiarles el valor, se las cambia a let. Pero no es obligación, también podés usar let, y usar const si sabés que es una constante.

    EJEMPLOS:

    var value = "some value";

let name = "john";

const lastName = "jordan";

    Comillas dobles o simples

    Podés usar las comillas que quieras.

    Pero hay algunos casos que tenes que usar una o la otra. O usar el signo \ (alt-gr º) antes de la comilla ' o ".

    Y para evitar todo esto se puede usar las comillas backticks `` (apretar dos veces la tecla que está a la derecha de la p).

    EJEMPLOS:

    const name = "john's courses are the best";

const name = 'john\'s courses are the best';

const saludo = 'hola a "TODOS" mis amigos';

    Concatenación de string

    combinar valores string

    Con las comillas backticks `` (apretar dos veces la tecla que está a la derecha de la p) es más fácil, pero lo explica después.

    EJEMPLOS:

    const name = "john"; 
const lastName = "smith"; 
let fullName = name + " " + lastName; 
console.log("Hello " + fullName);

const website = "google"; 
const url = "https://www." + website + ".com"; 
console.log(url);

    Números

    Javascript es Loosely typed. Es decir que no hay que declarar el type a la variable. Javascript lo hace automáticamente basándose en el valor que le pasamos.

    En la consola, los números aparecen en azul. Y en cambio los strings aparecen en negro. Si recibís un número que viene de data externa, por ejemplo de un imput, puede figurar como un string. En la consola va a estar en negro.

    EJEMPLOS:

    const entero = 34; 
const decimal = 2.466; 
const externalNumber = "2.466"; 
console.log(entero); 
console.log(decimal);
console.log(externalNumber);

const number1 = 30; 
const number2 = 20; 
const add = number1 + number2; 
const sub = number1 - number2; 
const mult = number1 * number2; 
const div = number1 / number2; 
console.log(add);
console.log(sub); 
console.log(mult); 
console.log(div);

    Operadores

    Los operadores +=, -=, /=, *=, le suman, o restan, o dividen, o multiplican al valor ya existente en la variable.

    Los operadores ++, --, le suman 1, o le restan 1, al valor existente en la variable.

    El operador % (módulo, modulus) devuelve el número restante de una división a números enteros.

    EJEMPLOS:

    let number = 40; 
number += 5; 
console.log(number); // va a ser 45 
number++; 
console.log(number); // le suma 1, va a ser 46

const pizzaSlices = 10;
const people = 3; 
const amount = pizzaSlices % people; 
console.log(amount); // queda una porción de pizza
